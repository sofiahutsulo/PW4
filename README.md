# PW4
Навчальний Android-застосунок мовою Kotlin для розрахунку параметрів електричної мережі та струмів короткого замикання.

## Опис
У межах практичної роботи №4 я реалізувала мобільний застосунок, який автоматизує розрахунок вибору кабелю та струмів короткого замикання для електричних мереж.

## Технології
- Kotlin  
- Android Studio
